# 🎮 iDotMatrix Web Controller
# متحكم شاشة البكسل

تطبيق ويب للتحكم بشاشات البكسل iDotMatrix عبر البلوتوث من المتصفح.

## 🚀 التشغيل

### الطريقة 1: فتح مباشر
افتح `index.html` في متصفح **Chrome** أو **Edge**

> ⚠️ ميزة البلوتوث تتطلب HTTPS أو localhost

### الطريقة 2: سيرفر محلي (مُوصى به)
```
npx -y http-server . -p 8888
```
ثم افتح: http://localhost:8888

## 📋 المتطلبات
- متصفح **Chrome** أو **Edge** (يدعم Web Bluetooth)
- بلوتوث مفعّل في الجهاز
- شاشة بكسل **iDotMatrix** (16×16 أو 32×32)

## 🎯 الميزات
| الميزة | الوصف |
|--------|-------|
| 🔗 الاتصال | بحث وتوصيل تلقائي عبر بلوتوث |
| 🕐 الساعة | 8 أنماط + لون + تاريخ |
| 🎨 الرسم | كانفس بكسل تفاعلي (قلم، ممحاة، تعبئة) |
| 📝 النص | 9 أنماط حركة + ألوان |
| 🖼️ الصور | سحب وإفلات + تصغير تلقائي |
| 🏆 النتائج | لوحة نتائج بأزرار ± |
| ⏱️ المؤقت | عد تنازلي + ستوب واتش |
| ⚙️ الإعدادات | سطوع، شاشة، وقت، إعادة ضبط |

## 📁 الملفات
```
index.html   ← الصفحة الرئيسية
style.css    ← التصميم
ble.js       ← اتصال البلوتوث
protocol.js  ← أوامر الجهاز
app.js       ← منطق الواجهة
```
